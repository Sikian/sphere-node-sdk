    Class ChannelService

    Defined in: src/coffee/services/channels.coffee
    Inherits: BaseService

    Overview

    Define a ChannelService to interact with the HTTP channels endpoint.

    Channels represent a source or destination of different entities.

    Examples:

    service = client.channels
service.byId('123').fetch()
.then (result) ->
  service.byId('123').update
    version: result.body.version
    actions: [
      {
        action: 'changeName'
        name:
          en: 'Foo'
      }
    ]

    Promise ::ensure(key, roles) Source

    Ensure a channel exists. It tries fetching one with given key, if not found it will create one otherwise will update it. Given roles are also ensured to exist.

    Examples:

    service = client.channels
roles = ['InventorySupply', 'OrderImport']
service.ensure('foo', roles)

    Parameters:

    • (String) key — String A unique identifier for channel within the project
    • (Array) roles — Array A list of ChannelRole the channel must have

    Throws:

    • (Error) — Throws an Error if key and roles are missing

    Returns:

    • (Promise) — a Promise, fulfilled with an Object or rejected with an instance of an Error
