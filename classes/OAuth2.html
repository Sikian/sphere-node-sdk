    Class OAuth2

    Defined in: src/coffee/connect/oauth2.coffee

    Overview

    Define an HTTP client to connect to the SPHERE.IO OAuth webservice.

    Examples:

    oa = new OAuth2
  config:
    client_id: "CLIENT_ID_HERE"
    client_secret: "CLIENT_SECRET_HERE"
    project_key: "PROJECT_KEY_HERE"
  host: 'auth.sphere.io' # optional
  accessTokenUrl: '/oauth/token' # optional
  timeout: 20000 # optional
  rejectUnauthorized: true # optional
  user_agent: 'sphere-node-connect' # optional
oa.getAccessToken (error, response, body) -> # do something

    Parameters:

    • (Object) options — An Object to configure the service
      • (Object) config — Object It contains the credentials project_key, client_id, client_secret
      • (String) host — String The host (default auth.sphere.io)
      • (String) protocol — String The protocol (default https)
      • (String) accessTokenUrl — String The url path to the token endpoint (default /oauth/token)
      • (String) user_agent — String The request User-Agent (default sphere-node-connect)
      • (Number) timeout — Number The request timeout (default 20000)
      • (Boolean) rejectUnauthorized — Boolean Whether to reject clients with invalid certificates or not (default true)
