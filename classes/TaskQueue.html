    Class TaskQueue

    Defined in: src/coffee/task-queue.coffee

    Overview

    A TaskQueue allows to queue tasks (that return Promises) which will be executed in parallel until a max limit, meaning that new tasks will not be triggered until the previous ones are resolved.

    Examples:

    callMe = ->
  new Promise (resolve, reject) ->
    setTimeout ->
      resolve true
    , 5000
task = new TaskQueue maxParallel: 50 # default 20
task.addTask callMe
.then (result) -> # result == true
.catch (error) ->
